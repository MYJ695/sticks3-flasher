推荐使用桌面版 Chrome 或 Edge。把 StickS3 用 USB 接到当前电脑后，点击目标固件按钮，在浏览器弹窗中选择串口即可。

  1. 1
    连接 StickS3；如果之前连接失败，先重新插拔 USB。
  2. 2
    点击目标固件的烧录按钮，并在弹窗里选择 StickS3 串口。
  3. 3
    保持页面打开，等待写入完成和设备重启。

请先用 USB 连接 StickS3，然后选择一个固件烧录。Chrome / Edge 需要 HTTPS 页面；当前 GitHub Pages 地址可直接使用。
注意：浏览器安全限制无法绕过“选择串口”这个系统弹窗，但本页面会把弹窗之后的连接、下载、写入和重启合并为自动流程。若连接失败，关闭其他串口工具，重新插拔 USB；必要时按住 Boot 键再点击烧录。
