StickS3 一键烧录

选择固件后，只需要点一次按钮并在浏览器弹窗里选择串口；页面会自动连接、下载 bin、写入 Flash、重启设备。

烧录状态等待操作
请先用 USB 连接 StickS3，然后选择一个固件烧录。Chrome / Edge 需使用 http://127.0.0.1:8000/ 或 HTTPS 页面。
  1. 插上 StickS3，点击目标固件的 选择串口并烧录。
  2. 浏览器弹窗中选择 StickS3 串口后，后续步骤会自动完成。
  3. 若连接失败，关闭串口监视器，重新插拔 USB；必要时按住 Boot 键再点按钮。

注意：浏览器安全限制无法绕过“选择串口”这个弹窗，但可以把弹窗之后的多次确认和 Connect/Install 点击合并为自动流程。
